
# Live collaboration whiteboard

## built using Reactjs, nodejs, express

### Features
1. Draw lines
2. Draw rectangles
3. Freehand drawing
4. undo 
5. Redo
6. Live collaboration (support 2 people as of now)

#### libraries used
[roughjs](https://www.npmjs.com/package/roughjs)

[perfect-freehand](https://www.npmjs.com/package/perfect-freehand)

[ws](https://www.npmjs.com/package/ws)
